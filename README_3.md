# AI Recommendation Logic – Tech Stack Recommender

## Overview

This project is a recommendation system that suggests career paths based on user skills and interests. It uses TF-IDF Vectorization and Cosine Similarity to match user preferences with relevant technology roles.

## Features

* User Skill Input
* TF-IDF Feature Extraction
* Cosine Similarity Matching
* Career Recommendation Engine
* Top 3 Recommended Career Paths

## Technologies Used

* Python
* Scikit-Learn

## Example Input

Python

Cloud

Automation

## Example Output

1. DevOps Engineer
2. Cloud Architect
3. Backend Developer

## How to Run

```bash
python tech_stack_recommender.py
```
